Action Code Barcodes

Print these barcodes and place them at your workstation for scanner-only operation

Action Codes

MOVE

Move item to another box

ACTION:MOVE

ADD

Add quantity to item
or add item to order

ACTION:ADD

TAKE

Remove quantity from item

ACTION:TAKE

DONE

Pack order / confirm action

ACTION:DONE

CANCEL

Cancel current action
or cancel order

ACTION:CANCEL

Type Codes (Create New)

TYPE: BOX

Create new box from barcode

TYPE:BOX

TYPE: ITEM

Create new item from barcode

TYPE:ITEM

TYPE: ORDER

Create new order from barcode

TYPE:ORDER

Item Operations

  1. Scan an ITEM barcode - Item becomes "selected"
  2. Scan an ACTION code:
    • ACTION:MOVE → Scan target BOX to move item
    • ACTION:ADD → Enter quantity to add to inventory
    • ACTION:TAKE → Enter quantity to remove from inventory
  3. Scan ACTION:CANCEL or press ESC to cancel

Order Operations

  1. Scan an ORDER barcode - Order becomes "selected"
  2. Scan ACTION:ADD - Now scan item barcodes to add to order
  3. Enter quantity when prompted (removes from inventory)
  4. Scan ACTION:DONE to pack the order
  5. Scan ACTION:CANCEL to cancel the order

Creating New Items

  1. Scan an unknown barcode - Prompt appears
  2. Scan TYPE code: TYPE:BOX, TYPE:ITEM, or TYPE:ORDER
  3. Fill in details - Barcode is prefilled
  4. For items: Scan a box barcode to assign location
