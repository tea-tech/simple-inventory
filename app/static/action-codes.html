Action Code Barcodes

Print these barcodes and place them at your workstation for scanner-only operation

Operations (OP:)

MOVE

Move item to another box
or box to warehouse

OP:MOVE

ADD

Add quantity to item
or add item to package

OP:ADD

TAKE

Remove quantity from item
or item from package

OP:TAKE

CHANGE

Change the type
of an entity

OP:CHANGE

Actions (ACT:)

OK

Confirm / pack package

ACT:OK

CANCEL

Cancel current action
or cancel package

ACT:CANCEL

Type Codes (TYPE:)

BOX

Create new box from barcode

TYPE:BOX

ITEM

Create new item from barcode

TYPE:ITEM

PACKAGE

Create new package from barcode

TYPE:PACKAGE

Item Chains

  1. Scan an ITEM barcode - Item becomes selected
  2. Scan an operation code:
    • OP:MOVE → Scan target BOX to move item
    • OP:ADD → Enter quantity to add to inventory
    • OP:TAKE → Enter quantity to remove from inventory
  3. Scan ACT:CANCEL or press ESC to cancel

Box Chains

  1. Scan a BOX barcode - Box becomes selected
  2. Scan OP:MOVE → Select warehouse to move box
  3. Scan ACT:CANCEL or press ESC to cancel

Package Chains

  1. Scan a PACKAGE barcode - Package becomes selected
  2. Scan OP:ADD - Now scan item barcodes to add to package
  3. Enter quantity when prompted (removes from inventory)
  4. Scan ACT:OK to pack the package
  5. Scan ACT:CANCEL to cancel the package

Creating New Entities

  1. Scan an unknown barcode - Prompt appears
  2. Scan TYPE code: TYPE:BOX, TYPE:ITEM, or TYPE:PACKAGE
  3. Fill in details - Barcode is prefilled
  4. For items: Scan a box barcode to assign location
